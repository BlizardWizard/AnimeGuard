.setup prefix [prefix] --- sets the server's custom prefix
.setup upvote [emoji] --- sets the upvote emoji in the '.poll' command
.setup downvote [emoji] --- sets the downvote emoji in the '.poll' command

.setup rules [text] --- sets response for the '.rules' command
